**GHikari Toolbox** is a modern, web-based Android management suite. It combines powerful ADB/Fastboot capabilities with an elegant Aero glassmorphism UI, a built-in interactive terminal, and a Live2D-integrated AI assistant to provide the ultimate debugging experience.

## ✨ Features / 功能 / 特徴 / 주요 기능 / Особенности

- **💎 Aero Glass UI**: Stunning visual design with dynamic gradients, 3D card effects, and meteor animations. / 优雅的毛玻璃设计，支持动态渐变、3D 卡片动效与流星背景。
- **📟 Interactive Terminal**: ZSH-like web console with autocomplete (Tab), history, and real-time process control. / 类 ZSH 网页终端，支持 Tab 补全、历史记录与实时进程控制。
- **🤖 AI Assistant**: Integrated with Live2D character, supporting OpenAI-compatible APIs for smart command suggestions. / 深度集成 Live2D 看板娘，支持 OpenAI 兼容接口，提供智能命令建议。
- **⚡ Fastboot Pro**: Integrated flashing panel with automatic partition detection and one-click image flashing. / 内嵌式 Fastboot 刷写面板，支持自动分区识别与一键镜像刷写。
- **🛠️ Auto-StrapUp**: Zero-config environment setup. Automatically downloads ADB/Fastboot tools for your OS. / 零配置环境搭建，自动为您的操作系统下载 ADB/Fastboot 工具。

## 🚀 Quick Start / 快速开始 / クイックスタート / 빠른 시작 / Быстрый старт

### 1. Clone the repository / 克隆仓库
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. Run the Auto-Setup / 运行自动化配置
**GHikari** features an automatic environment setup script. Just run:
```bash
# Windows
start.bat
# or
python strap-up.py

# Linux / macOS
python3 strap-up.py
```
*The script will automatically create a venv, install dependencies, and download platform-tools if missing.*
*脚本将自动创建虚拟环境、安装依赖，并在工具缺失时自动下载 platform-tools。*
