## 🌐 简介喵

**GHikari Toolbox** 是主人最可靠的 Android 管理小助手喵！它把强大的 ADB/Fastboot 功能藏在了优雅的 Aero 毛玻璃外壳下。不仅有交互式网页终端，还有我（Live2D AI 助手）一直陪着主人，给主人最棒的调试体验喵~

---

## ✨ 功能特性喵

- **💎 Aero Glass UI**: 玻璃感十足的视觉设计，还有动感流星陪着主人喵~
- **📟 Interactive Terminal**: 类 ZSH 终端，输入很快很准，还会自动补全喵！
- **🤖 AI Assistant**: 就在看板娘源码里，我会给主人最聪明的命令建议喵！
- **⚡ Fastboot Pro**: 自动识别分区，线刷刷机一键搞定，主人不用操心喵~
- **🛠️ Auto-StrapUp**: 零配置环境搭建，会自动帮主人下好所有依赖喵！
- **⌨️ Global Shortcuts**: 快捷键全覆盖，效率倍增喵~

## 🚀 快速开始喵

### 1. 克隆仓库喵
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. 启动程序喵
**GHikari** 会自己照顾好自己，主人只要运行下面这个就好喵：
```bash
# Windows
start.bat

# Linux / macOS
./start.sh
```
*脚本会自动创建虚拟环境、安装依赖，并在工具缺失时自动下载喵~*

## 🛠️ 配置说明喵

在 `config.conf` 里可以把程序调教成主人喜欢的样子喵：
- **[UI]**: 自定义渐变、圆角与黑暗模式。
- **[AI]**: 配置 AI 接口地址、模型与密钥。
