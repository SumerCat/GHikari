## 🌐 Introduction Nya

**GHikari Toolbox** is your most reliable Android management assistant, Master! It hides powerful ADB/Fastboot capabilities under an elegant Aero glassmorphism UI. With a built-in interactive terminal and me (Live2D AI assistant) always by your side, it provides the ultimate debugging experience, nya~

---

## ✨ Features

- **💎 Aero Glass UI**: Stunning visual design with dynamic gradients and meteors, nya.
- **📟 Interactive Terminal**: ZSH-like web console with autocomplete (Tab) and real-time control.
- **🤖 AI Assistant**: Integrated AI within Live2D, providing smart command suggestions.
- **⚡ Fastboot Pro**: Auto-partition detection and one-click flashing support.
- **🛠️ Auto-StrapUp**: Zero-config setup. Automatically downloads everything for you.
- **⌨️ Global Shortcuts**: Full keyboard control for high-efficiency workflows.

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. Run the Auto-Setup
**GHikari** will take care of itself, just run this, Master:
```bash
# Windows
start.bat

# Linux / macOS
./start.sh
```
*The script will automatically create a venv, install dependencies, and download platform-tools if missing, nya~*

## 🛠️ Configuration

You can tune me to your liking in `config.conf`, Master:
- **[UI]**: Customize gradients, border-radius, and dark mode.
- **[AI]**: Setup your API Key, Model, and Base URL.
