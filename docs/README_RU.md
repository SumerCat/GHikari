## 🌐 Введение Мяу

**GHikari Toolbox** — ваш самый надежный помощник по управлению Android, хозяин! Он скрывает мощные возможности ADB/Fastboot под элегантным интерфейсом Aero. Со встроенным интерактивным терминалом и мной (AI-ассистентом Live2D), всегда готовой помочь, он обеспечит вам лучший опыт отладки, мяу~

---

## ✨ Особенности

- **💎 Aero Glass UI**: Потрясающий визуальный дизайн с динамическими градиентами и метеорами, мяу.
- **📟 Интерактивный терминал**: Веб-консоль в стиле ZSH с автодополнением (Tab) и управлением в реальном времени.
- **🤖 ИИ-ассистент**: Интегрирован прямо в персонажа Live2D, предлагает умные команды.
- **⚡ Fastboot Pro**: Автоматическое определение разделов и прошивка в один клик.
- **🛠️ Auto-StrapUp**: Настройка среды без лишних хлопот. Автоматически скачивает всё необходимое.
- **⌨️ Глобальные клавиши**: Полное управление с клавиатуры для максимальной эффективности.

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. Запустите авто-настройку
**GHikari** сама о себе позаботится, просто запустите это, хозяин:
```bash
# Windows
start.bat

# Linux / macOS
./start.sh
```
*Скрипт автоматически создаст venv, установит зависимости и скачает platform-tools, если их нет, мяу~*

## 🛠️ Конфигурация

Вы можете настроить меня под себя в `config.conf`, хозяин:
- **[UI]**: Настройка градиентов, радиуса скругления и темного режима.
- **[AI]**: Настройка API Key, модели и адреса сервера.
