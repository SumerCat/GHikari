## 🌐 소개냥

**GHikari Toolbox**는 주인님의 가장 신뢰할 수 있는 Android 관리 도우미냥! 강력한 ADB/Fastboot 기능을 우아한 Aero 글래스모피즘 UI 아래에 숨겼다냥. 대화형 웹 터미널과 항상 주인님 곁에 있는 저(Live2D AI 어시스턴트)가 최고의 디버깅 경험을 제공할 거다냥~

---

## ✨ 주요 기능

- **💎 Aero Glass UI**: 유리 질감이 살아있는 비주얼 디자인과 역동적인 유성이 주인님과 함께한다냥~
- **📟 Interactive Terminal**: ZSH 스타일 터미널로 빠르고 정확하게 입력하고 자동 완성까지 지원한다냥!
- **🤖 AI Assistant**: Live2D 캐릭터 안에 통합되어 주인님께 똑똑한 명령어를 추천해 준다냥!
- **⚡ Fastboot Pro**: 파티션을 자동으로 인식하고 한 번의 클릭으로 플래싱을 완료한다냥!
- **🛠️ Auto-StrapUp**: 설정 없이 환경을 구축하고 필요한 모든 의존성을 자동으로 다운로드한다냥!
- **⌨️ Global Shortcuts**: 단축키로 작업 효율을 극대화할 수 있다냥~

## 🚀 빠른 시작

### 1. 저장소 클론
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. 자동 설정 실행
**GHikari**는 스스로를 잘 돌본다냥, 주인님은 아래 명령어만 실행하면 된다냥:
```bash
# Windows
start.bat

# Linux / macOS
./start.sh
```
*스크립트가 자동으로 가상 환경을 만들고, 의존성을 설치하며, 도구가 없으면 자동으로 다운로드한다냥~*

## 🛠️ 설정

`config.conf`에서 주인님이 원하는 대로 저를 길들일 수 있다냥:
- **[UI]**: 그라데이션, 모서리 곡률, 다크 모드 커스터마이징.
- **[AI]**: AI API 주소, 모델, 키 설정.
