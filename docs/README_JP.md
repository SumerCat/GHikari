## 🌐 紹介だにゃ

**GHikari Toolbox** は、ご主人様の最も信頼できる Android 管理アシスタントだにゃ！強力な ADB/Fastboot 機能をエレガントな Aero グラスモーフィズム UI の下に隠しているにゃ。インタラクティブな Web ターミナルと、いつもご主人様のそばにいる私（Live2D AI アシスタント）が、最高のデバッグ体験を提供するにゃ〜

---

## ✨ 特徴

- **💎 Aero Glass UI**: ガラス感たっぷりのビジュアルデザイン、動く流星も一緒だにゃ〜
- **📟 Interactive Terminal**: ZSH風ターミナル、入力が早くて正確、オートコンプリートもバッチリだにゃ！
- **🤖 AI Assistant**: 看板娘のソースコードに統合されているにゃ、賢いコマンド提案もお任せだにゃ！
- **⚡ Fastboot Pro**: パーティションを自動認識、一括フラッシュもワンクリックで完了だにゃ、心配いらないにゃ〜
- **🛠️ Auto-StrapUp**: 設定不要で環境構築、必要な依存関係は全部自動でダウンロードするにゃ！
- **⌨️ Global Shortcuts**: ショートカットキーも充実、効率倍増だにゃ〜

## 🚀 クイックスタート

### 1. リポジトリをクローン
```bash
git clone https://github.com/izumikori/FGHikari.git
cd FGHikari
```

### 2. 自動セットアップを実行
**GHikari** は自分で自分の面倒を見れるにゃ、ご主人様はこれを実行するだけでいいにゃ：
```bash
# Windows
start.bat

# Linux / macOS
./start.sh
```
*スクリプトは自動的に仮想環境を作成し、依存関係をインストールし、ツールが足りない場合は自動でダウンロードするにゃ〜*

## 🛠️ 設定

`config.conf` で、私をご主人様の好みに調教できるにゃ：
- **[UI]**: グラデーション、角丸、ダークモードをカスタマイズ。
- **[AI]**: AI APIのベースURL、モデル、キーを設定。
